What is lllibrary?

A living profile for every piece of land

lllibrary combines landowner knowledge with open data to create a Land Passport — a shareable, evolving portrait of any property.

✍

Guided Onboarding

A step-by-step intake walks you through contributing your property data — land use, soil, water, ecology, challenges and goals.

☂

Open Data Layer

For each property location, we auto-aggregate weather, soil type, bioregion classification, fire risk, elevation and more from public sources.

⚛

Land Passport

Each property gets a single, shareable page that displays owner-contributed data alongside auto-aggregated open data and seasonal guidance.
